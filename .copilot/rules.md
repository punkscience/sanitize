# General Directives

- Always use Go and Cobra Command library for command-line or terminal applications
- Use Bubble Tea as the interface library and architecture for terminal ui implementation
- Use SOLID principles in software development
- Comment code well
- Always write tests to ensure app functionality works as expected
- Make the test scripts part of the github CI/CD pipeline
- Whenever implementing features listed in the tech-spec.md, ensure you mark it completed when it is finished. 
- Always refer to tech-spec.md to know what the next tasks are.
